# Query Opportunities

Search `Opportunities/` plus linked Company, Customer, People, and Project notes. Use the configured workspace
playbook to find portfolio maps and stale thresholds.

Answer from records and cite each material claim with a wikilink. Distinguish recorded facts from inference,
and name missing data instead of deriving value or probability from stage. Query is read-only unless the user
separately requests a lifecycle mutation.

For portfolio hygiene, stale pursuits, or gate audits, route to the explicit `review-opportunities` skill
when it is installed; this command answers questions but does not impersonate a review.
