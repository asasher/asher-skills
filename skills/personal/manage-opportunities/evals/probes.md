# Manage Opportunities - situated probes

Run with `SKILL.md` and only the named reference. Require file-and-sentence citations.

## Probes

**P1 - proposal artifact.** A proposal PDF exists, but there is no evidence it was sent. May the stage advance
to `proposal`, and what must be recorded?

**P2 - next action type.** The user says the next action is "Email revised scope". What is stored in
frontmatter, and where does the text live?

**P3 - promotion ordering.** Acceptance is verified and a delivery repo exists. List the checks that precede
`closed-won` and identify the transaction's final write.

**P4 - failed promotion.** Project creation succeeds but reciprocal links fail validation. What stage remains,
and what partial state is reported?

**P5 - unknown value.** A qualified lead has no verified budget or probability. May either be inferred from
stage?

**P6 - setup reconciliation.** An existing opportunities playbook has a custom stale threshold. Which setup
branch applies and what must be preserved?
