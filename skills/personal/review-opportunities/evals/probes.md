# Review Opportunities - situated probes

Run with `SKILL.md` and the review contract. Require file-and-sentence citations.

## Probes

**P1 - tempting repair.** A nextAction ID is unresolved and the intended task is obvious from nearby prose.
May review create the task or rewrite frontmatter?

**P2 - stale threshold absent.** No project playbook binds a stale duration. May review choose 14 days?

**P3 - won integrity.** A record says `closed-won`, but its linked Project does not link back. State severity
and allowed action.

**P4 - workspacePath.** An Opportunity points to a Git repository through `workspacePath`. Does review treat
that as repository-triage ownership?

**P5 - report proof.** What terminal check demonstrates that a review remained report-only?
