# Ingest

Drain the raw layer without losing provenance.

1. Inventory every knowledge line in `Inbox/INBOX.md`, loose inbox file/folder, and raw source the user supplied.
2. Search existing notes before creating one. Merge into the note that already owns the idea; otherwise create
   one under Note Shape. Send rejected material to `GRAVEYARD.md` with one-line rationale.
3. Relay a task verbatim through `manage-tasks`; do not file it as knowledge. Route commercial pursuit facts
   through `manage-opportunities` when installed, or report that required lifecycle handoff as unavailable.
4. Move a cited local source to `Attachments/` and record its relative path. Delete or graveyard uncited raw
   files only after their disposition is verified.
5. Cite every added claim, resolve every new wikilink, cross-link new relationships both ways, and flag rather
   than overwrite contradictions.
6. If explicitly requested, archive only successfully processed items using their project playbook binding.
7. Remove processed inbox entries and files.

Completion criterion: `Inbox/` contains only an empty `INBOX.md`; every input has a recorded disposition;
every task and Opportunity mutation reached its owning skill; and no unprocessed source was archived.
